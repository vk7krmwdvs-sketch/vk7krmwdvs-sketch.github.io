Not connected
ARM

Open in Bluefy, select w33 Control Hub, and disconnect LightBlue first. Setup and calibration are saved on this device.

Robot setup

Assign the left motor, swing arm, and right motor to the hub outputs. You can use the calibration wizard or set them manually.

Manual setup

Each part must use a different output.

Calibration wizard

Find the left motor

Tap each output until the left wheel moves. Keep the rover lifted.

Tap the same button again to stop that test.

Find the right motor

Tap the output that moves the right wheel.

Find the swing arm

Tap the output that moves the arm.

Motor direction

Use these toggles so pushing the joystick forward drives both wheels forward.

Press again to stop.
